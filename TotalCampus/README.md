# TotalCampus

Companion resources for **TotalCampus**, the native macOS + iPadOS app that
teaches [Total.js](https://www.totaljs.com) — backend (Total.js 5) and frontend
(jComponent v20) — by building a real app called **Tasker**.

## Contents

- **[`tasker/`](./tasker/)** — the **complete Tasker app**, a runnable Total.js 5
  reference project. Every file is commented with the **module that taught it**,
  so you can build along in your own editor and compare. See
  [`tasker/README.md`](./tasker/README.md) for the lesson map and how to run it.
- **[`languages/`](./languages/)** — the importable **language pack template** to
  translate the whole app (interface + course) into your language. See
  [`languages/README.md`](./languages/README.md).

## About the app

TotalCampus takes you from zero — even if you've never programmed — to a real
Total.js developer: accounts, validation, databases, real-time, reminders, an API
and a jComponent UI, all built step by step. The course runs code safely inside
the app; this repository gives you the finished project to keep as a reference.

Product page: [totaldesktop.totaljs.es](https://totaldesktop.totaljs.es)
